# Presentasi & Laporan

**Persona yang digunakan**: ASN Assistant 🤖

---

## Tentang

Persona **ASN Assistant** adalah asisten umum yang membantu menyusun presentasi, laporan berbasis data, dan visualisasi informasi untuk kebutuhan kedinasan.

## Contoh Penggunaan

### 1. Kerangka Presentasi

```
Buat kerangka presentasi untuk sosialisasi aplikasi SIPD
kepada 50 ASN di lingkungan Pemkab Aceh Tengah.
Durasi 30 menit, 10 slide.
```

### 2. Laporan Berbasis Data

```
Saya punya data: 150 layanan publik di tahun 2025,
85% puas, 10% cukup puas, 5% tidak puas.
Buat analisis dan rekomendasi perbaikannya.
```

### 3. Infografis Narasi

```
Bantu saya menyusun narasi untuk infografis capaian
Diskominfo Aceh Tengah tahun 2025. Capaian:
- 100 desa terhubung internet
- 50 ASN terlatih IT
- 25 aplikasi layanan publik
```

### 4. Executive Summary

```
Buat executive summary dari laporan kinerja triwulan I
Diskominfo Aceh Tengah. Panjang 1 halaman.
```

## Tips

- Sertakan data konkret dan angka untuk hasil lebih akurat
- Minta draft, lalu reviu dan minta revisi jika perlu
- Gunakan untuk membuat outline, lalu kembangkan sendiri
