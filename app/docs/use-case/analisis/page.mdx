# Analisis Kebijakan

**Persona yang digunakan**: Analis Kebijakan 📊

---

## Tentang

Persona **Analis Kebijakan** membantu ASN dalam menganalisis peraturan, kebijakan, dan regulasi dengan pendekatan terstruktur.

## Contoh Penggunaan

### 1. Ringkasan Peraturan

```
Ringkas UU Nomor 27 Tahun 2022 tentang Pelindungan Data Pribadi.
Fokus pada kewajiban instansi pemerintah.
```

### 2. Perbandingan Kebijakan

```
Bandingkan Permenkominfo No. 5/2021 dengan Permenkominfo No. 3/2023
tentang penyelenggara sistem elektronik. Apa saja perubahan utamanya?
```

### 3. Analisis Dampak

```
Analisis dampak dari penerapan Peraturan Presiden No. 95/2018
tentang SPBE terhadap unit IT di Diskominfo. Identifikasi tantangan
dan rekomendasi.
```

### 4. Pro-Con Analysis

```
Berikan analisis pro dan kontra dari kebijakan work from home (WFH)
untuk ASN di lingkungan pemerintahan daerah.
```

## Metode Analisis

- **SWOT** — Strengths, Weaknesses, Opportunities, Threats
- **Pro-Con** — Kelebihan dan kekurangan
- **Perbandingan** — Before-after, cross-regulation
- **Rekomendasi** — Best practice dan langkah tindak lanjut
