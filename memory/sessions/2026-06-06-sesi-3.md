## 🧠 Keputusan & Catatan Penting

1. **Tailwind v4 dark mode**: Menggunakan `@custom-variant dark (&:where(.dark, .dark *))` untuk class-based toggling
2. **FOUC prevention**: Inline script di `<head>` untuk set class sebelum render
3. **Platform binary fix**: Project sebelumnya diinisialisasi di Windows, perlu `rm -rf node_modules && npm install` di macOS untuk binary yang cocok
4. **Routing username**: Sementara pakai static object, siap untuk integrasi database
